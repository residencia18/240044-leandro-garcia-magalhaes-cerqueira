Formulário do Usuário

Username
* O campo não deve conter espaços
* Máx. de caracteres: 12
* O campo não pode ser vazio
Senha
* A senha deve conter no mínimo 4 caracteres
* A senha deve incluir pelo menos uma letra maiúscula ex:(A,F,Z...)
* A senha deve incluir pelo menos um simbolo ex:(*,&..)
Email
* Insira um e-mail válido
Nome completo
* Insira um nome e sobrenome
Telefone
* Insira um número de telefone válido (xx xxxx-xxxx ou (xx) xxxx-xxxx)
Endereço
* Insira um endereço válido (Ex: Rua A, 100, Bairro B, Cidade C - Estado ES)
Data de nascimento
* Insira uma data de nascimento válida (dd/mm/aaaa)
* O usuário deve possuir mais de 18 anos
Gênero
* O campo não pode ser vazio
Profissão
* O campo não pode ser vazio
